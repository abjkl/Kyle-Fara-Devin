# 开始使用

## 前提条件
- Node.js 16 或更高版本
- 云服务提供商账户（Azure、Google Cloud 或 Vultr）
- 基本的 SSR 概念理解

## 安装
```bash
npm install ssr-deploy-docs
```

## 快速开始
1. 配置云服务提供商凭证
2. 创建配置文件
3. 运行部署命令

## 云服务提供商设置
### Azure
1. 创建 Azure 账户
2. 设置服务主体
3. 配置环境变量

### Google Cloud
1. 创建 Google Cloud 账户
2. 设置服务账户
3. 下载凭证文件

### Vultr
1. 创建 Vultr 账户
2. 生成 API 密钥
3. 设置环境变量
