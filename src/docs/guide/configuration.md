# Configuration Guide

Learn how to configure your SSR server for optimal performance and security.

## Basic Configuration

### Server Location
Choose the server location closest to your target audience:
- Asia: Singapore, Japan, Korea
- Europe: Germany, Netherlands
- Americas: Los Angeles

### Server Specifications
Recommended specifications for different usage scenarios:
- Small: 1 CPU, 1GB RAM
- Medium: 2 CPU, 2GB RAM
- Large: 4 CPU, 4GB RAM
